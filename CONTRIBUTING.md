# Contributing to Fanpy

Thank you for your interest in contributing to Fanpy!

## Reporting Issues

Please open an issue on GitHub describing the problem, including:
- Your Home Assistant version
- Steps to reproduce
- Relevant logs

## Submitting Changes

1. Fork the repository.
2. Create a feature branch.
3. Make your changes and commit.
4. Push to your fork and open a pull request.

## Code Style

- Follow Python PEP 8.
- Use type hints where possible.
- Keep functions focused and well-documented.
