# Fanpy

Fanpy Pro is a custom integration for Home Assistant that configures ceiling fans for use with the Fan Custom Card Lovelace card.

## Features
- Multi-step setup wizard (area, mode, features, Broadlink)
- Creates binary_sensor entities for card more-info
- Generates complete YAML configuration (input_boolean, input_select, templates)
- Generates all Broadlink RF scripts
- Supports 3 languages: English, Spanish, Catalan

For more information, check the [README](https://github.com/figorr/fanpypro).
